Detailed Description

This class is instanced by the TracerUpdateSenderPlugin and is run in a subthread.

The class listens to SYNC messages from DataHub and the other TRACER clients, keeps the timestamps in check, and notifies TracerUpdateSenderPlugin of the arrival of the message

Main loop, executes all operations.
