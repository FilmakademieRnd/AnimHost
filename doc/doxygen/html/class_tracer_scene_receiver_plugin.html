enum TracerSceneReceiverPlugin::NodeType

Enumeration listing possible TRACER scene node types.

EnumValue Int Value Description
GROUP 0 Generic node
GEO 1 Node representing a geometry (more accurate description to follow)
LIGHT 2 Node representing a light (more accurate description to follow)
CAMERA 3 Node representing a virtual camera (more accurate description to follow)
SKINNEDMESH 4 Node representing a skinned mesh, a subnode of a character that contains information about the skeleton (more accurate description to follow)
CHEARACTER 5 Node representing a character (for now, equivalent in content to the generic node)

TracerSceneReceiverPlugin::TracerSceneReceiverPlugin ( )

Default constructor.

Instantiates the scene receiver class, moves it to its separate thread and connects the various signals to the associated functions:

  • SceneReceiver::passCharacterByteArray() signal is connected to TracerSceneReceiverPlugin::processCharacterByteData()
  • SceneReceiver::passSceneNodeByteArray() signal is connected to TracerSceneReceiverPlugin::processSceneNodeByteData()
  • SceneReceiver::passHeaderByteArray() signal is connected to TracerSceneReceiverPlugin::processHeaderByteData()
  • TracerSceneReceiverPlugin::requestCharacterData() signal is connected to SceneReceiver::requestCharacterData()
  • TracerSceneReceiverPlugin::requestSceneNodeData() signal is connected to SceneReceiver::requestSceneNodeData()
  • TracerSceneReceiverPlugin::requestHeaderData() signal is connected to SceneReceiver::requestHeaderData()

void TracerSceneReceiverPlugin::onButtonClicked ( )
privateslot

Slot called when the "Request Data" button is clicked.

Trigger the request of header, scene nodes and character data. This starts the feedback-loop of requesting, receiving, processing, and updating the connected application plugins

requestSceneNodeData() is then called after processCharacterData() is done (i.e. at thew end of its body)

void TracerSceneReceiverPlugin::processCharacterByteData ( QByteArray * charByteArray)
privateslot

Processing the character byte sequence received from the external TRACER client.

Unpacking the received byte array representing the characters (in the format of Character Packages) in the scene and filling a CharacterObjectSequence with these data

"Bookmark" for reading the character byte array and skipping uninteresting sequences of bytes
