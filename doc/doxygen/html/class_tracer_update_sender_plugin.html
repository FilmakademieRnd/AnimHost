TracerUpdateSenderPlugin::TracerUpdateSenderPlugin ( )

Default constructor.

Instantiates message sender and tick receiver classes, moves both to their separate threads and connects the various signals to the associated functions:

  • QThread::started() signal is connected to AnimHostMessageSender::run()
  • QThread::started() signal is connected to TickReceiver::run()
  • TickReceiver::tick() signal is connected to TracerUpdateSenderPlugin::ticked()
