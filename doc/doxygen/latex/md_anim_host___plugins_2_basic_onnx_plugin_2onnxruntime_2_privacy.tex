No data collection is performed when using your private builds built from source code.\hypertarget{md_anim_host___plugins_2_basic_onnx_plugin_2onnxruntime_2_privacy_autotoc_md3}{}\doxysubsection{\texorpdfstring{Official Builds}{Official Builds}}\label{md_anim_host___plugins_2_basic_onnx_plugin_2onnxruntime_2_privacy_autotoc_md3}
ONNX Runtime does not maintain any independent telemetry collection mechanisms outside of what is provided by the platforms it supports. However, where applicable, ONNX Runtime will take advantage of platform-\/supported telemetry systems to collect trace events with the goal of improving product quality.

Currently telemetry is only implemented for Windows builds and is turned {\bfseries{ON}} by default in the official builds distributed in their respective package management repositories (\href{../README.md\#binaries}{\texttt{ see here}}). This may be expanded to cover other platforms in the future. Data collection is implemented via \textquotesingle{}Platform Telemetry\textquotesingle{} per vendor platform providers (see \href{../onnxruntime/core/platform/telemetry.h}{\texttt{ telemetry.\+h}}).\hypertarget{md_anim_host___plugins_2_basic_onnx_plugin_2onnxruntime_2_privacy_autotoc_md4}{}\doxysubsubsection{\texorpdfstring{Technical Details}{Technical Details}}\label{md_anim_host___plugins_2_basic_onnx_plugin_2onnxruntime_2_privacy_autotoc_md4}
The Windows provider uses the \href{https://docs.microsoft.com/en-us/windows/win32/tracelogging/trace-logging-about}{\texttt{ Trace\+Logging}} API for its implementation. This enables ONNX Runtime trace events to be collected by the operating system, and based on user consent, this data may be periodically sent to Microsoft servers following GDPR and privacy regulations for anonymity and data access controls.

Windows ML and onnxruntime C APIs allow Trace Logging to be turned on/off (see \href{../README.md\#api-documentation}{\texttt{ API pages}} for details). For information on how to enable and disable telemetry, see \href{./C_API.md\#telemetry}{\texttt{ C API\+: Telemetry}}. There are equivalent APIs in the C\#, Python, and Java language bindings as well.
