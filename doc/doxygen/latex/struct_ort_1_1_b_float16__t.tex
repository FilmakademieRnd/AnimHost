\doxysubsection{Detailed Description}
bfloat16 (Brain Floating Point) data type 

It is necessary for type dispatching to make use of C++ API The type is implicitly convertible to/from uint16\+\_\+t. The size of the structure should align with uint16\+\_\+t and one can freely cast uint16\+\_\+t buffers to/from \doxylink{struct_ort_1_1_b_float16__t}{Ort\+::\+BFloat16\+\_\+t} to feed and retrieve data.
