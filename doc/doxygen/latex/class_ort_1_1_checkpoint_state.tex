\doxysubsection{Detailed Description}
Holds the state of the training session. 

This class holds the entire training session state that includes model parameters, their gradients, optimizer parameters, and user properties. The \doxylink{class_ort_1_1_training_session}{Ort\+::\+Training\+Session} leverages the \doxylink{class_ort_1_1_checkpoint_state}{Ort\+::\+Checkpoint\+State} by accessing and updating the contained training state. \begin{DoxyNote}{Note}
Note that the training session created with a checkpoint state uses this state to store the entire training state (including model parameters, its gradients, the optimizer states and the properties). The \doxylink{class_ort_1_1_training_session}{Ort\+::\+Training\+Session} does not hold a copy of the \doxylink{class_ort_1_1_checkpoint_state}{Ort\+::\+Checkpoint\+State} and as a result, it is required that the checkpoint state outlive the lifetime of the training session. 
\end{DoxyNote}

Load a checkpoint state from directory on disk into checkpoint\+\_\+state. 

This function will parse a checkpoint directory, pull relevant files and load the training state and return an instance of \doxylink{class_ort_1_1_checkpoint_state}{Ort\+::\+Checkpoint\+State}. This checkpoint state can then be used to create the training session by instantiating \doxylink{class_ort_1_1_training_session}{Ort\+::\+Training\+Session}. By doing so, the training session will resume training from the given checkpoint state.
